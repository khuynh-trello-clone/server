### Full Stack MERN Project - Trello Clone App

A base project structure for building a Trello clone application using the MERN stack (MongoDB, Express, React, Node.js)

### Technical Requirements:

```shell
* nodejs >= 18.16.0
* npm = v9.8.1
* yarn = v1.22.19
```
